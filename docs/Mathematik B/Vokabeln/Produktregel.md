**1. Beispiel:** 
$\\f(x) = x^3 * x^5\\$

$⇒ g(x) = x^3, g’(x) = 3x^2\\$
$⇒ h(x) = x^5, h’(x) = 5x^4\\$

$⇒ f’(x) = g’(x) * h’(x) + g(x) h’(x)\\$
$⇒ 3x^2*x^5 + x^3*5x^4\\$
$⇒8x^7$
