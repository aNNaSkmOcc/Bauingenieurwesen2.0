# geometrische Reihe

- $\sum \limits_{n=0}^{\infty} q^k = 1+q^1+q^2+q^3+q^4+... = \frac{1}{1-q}$
- Wenn |q| < 1, ist die geometrische Reihe konvergent.
- Ansonsten ist auch diese divergent.
