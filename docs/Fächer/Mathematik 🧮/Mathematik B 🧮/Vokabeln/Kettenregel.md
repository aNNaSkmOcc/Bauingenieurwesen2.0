# Kettenregel


- Die Kettenregel, nutzt man bei Verkettungen von Funktionen.

![[IMG_8CB509CC7A90-1.jpeg|IMG_8CB509CC7A90-1.jpeg|500]]

---

**1.  Beispiel:**
$\\f(x) = (x^4 + 5)^2\\$


$⇒ g(x) = x^2 , g’(x) = 2*x$ (äußere Funktion)
$⇒ h(x) = x^4 + 5, h’(x) = 4x^3$ (innere Funktion)

$⇒ f’(x) = g’( h(x) )*h’(x)$\
$⇒2(x^4+5)*4x^3$\
$⇒8x^3(x^4+5)$
